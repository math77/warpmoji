# warpmoji
 
An onchain sticker protocol for decentralized social networks.


### Devlog

#### 04/18/2025

Drafting an initial version of the protocol's possible architecture and deciding what features it should have. Researching the best file types for storing the stickers.

The final project will probably consist of:
- Protocol deployed on base
- Webapp to create sets and upload/manage stickers
- API (easier querying the stickers) 
- SDK that allows searching for available stickers for an specific address/fid
